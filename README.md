# Smart Energy Management and Room Automation System

## Overview

The Smart Energy Management and Room Automation System is an Arduino Uno–based embedded system designed to improve energy efficiency, comfort, and safety in indoor environments.

The system automatically controls room appliances based on occupancy detection, ambient light intensity, temperature, and smoke levels using multiple sensors integrated with an Arduino Uno.

This project was developed as part of the "Idea Lab" course at RV College of Engineering (RVCE).

---

##  Features

- Automatic room occupancy detection using IR sensors
- Automatic lighting control using an LDR sensor
- Real-time temperature monitoring using a DHT11 sensor
- Smoke detection using an MQ-2 sensor
- Buzzer alert during hazardous conditions
- LCD display for live system status
- Energy-efficient appliance control
